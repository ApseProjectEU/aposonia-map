# APOSONÍA

Sound art installation by Gaetano Cappella investigating underwater acoustic pollution in Thermaikos Gulf, Thessaloniki.

**Funding**: Culture Moves Europe (Creative Europe / Goethe-Institut)
**Residency**: March–May 2026

## Features
- 4-language support (EN/IT/ES/EL)
- Interactive Leaflet.js map
- Museum-quality design
- Mobile-responsive

## Tech
- Pure HTML/CSS/JS
- Leaflet.js
- Canvas API
