# Aposonía Map

Interactive geolocation system for sound art performance along Nea Paralia, Thessaloniki.

## Quick Start

Open `index.html` in a browser. Map initializes centered on Nea Paralia.

## Features

- Click anywhere to get precise coordinates
- Create temporary markers (blue)
- Load points from `data/points.json` (red markers)
- Click markers to view details in sidebar
- Export temporary markers as JSON

## Technology

- Leaflet.js + OpenStreetMap
- Vanilla JavaScript
- No build process required
